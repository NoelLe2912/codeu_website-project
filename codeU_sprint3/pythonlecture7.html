Looping Through Strings

In Chapter 3, you learned to use for loops with the range function. Now you'll discover that you can loop through the characters in a string just as easily! This opens up powerful possibilities for text processing.

Basic String Loop

You can use a for loop to access each character in a string, one at a time: 

for letter in 'Gadsby':
    print(letter, end=' ')

# Output: G a d s b y

The Loop Variable

The variable defined in a for loop (like letter above) is called the loop variable. It's good practice to give it a descriptive name that makes your code easier to understand. 

# Less clear
for i in 'Python':
    print(i)

# More clear
for character in 'Python':
    print(character)

Checking for Specific Letters

By combining loops with if statements, you can search for specific characters: 

for letter in "Gadsby":
    if letter == 'E' or letter == 'e':
        print('This word has an "e"')

💡 Fun Fact: In 1939, Ernest Vincent Wright published a 50,000-word novel called "Gadsby" without using the letter "e" even once—quite a challenge since "e" is the most common letter in English!

Creating a Letter Checker Function

Let's turn our letter-checking code into a reusable function. We'll start simple and gradually improve it.

Step 1: Basic Function

First, let's encapsulate the loop in a function: 

def has_e():
    for letter in "Gadsby":
        if letter == 'E' or letter == 'e':
            print('This word has an "e"')

Step 2: Return a Value

Let's make it a pure function that returns True or False: 

def has_e():
    for letter in "Gadsby":
        if letter == 'E' or letter == 'e':
            return True
    return False

Notice the two return statements: one inside the loop (when we find an "e") and one outside (when we don't find any).

Step 3: Add a Parameter

Now let's make it work with any word by adding a parameter: 

def has_e(word):
    for letter in word:
        if letter == 'E' or letter == 'e':
            return True
    return False

# Test it
has_e('Gadsby')  # Returns: False
has_e('Emma')    # Returns: True

🎯 Key Pattern: When searching for something in a sequence, return True as soon as you find it. Only return False after checking everything.

Reading Word Lists from Files

To analyze many words, we need to read them from a file. Python makes this easy with file objects and methods.

Opening Files

Use the built-in open function to create a file object: 

file_object = open('words.txt')

# Read one line
file_object.readline()
# Returns: 'aa\n'

The \n represents a newline character that separates words in the file.

Methods vs Functions

readline is a method—a function associated with an object. You call it using the dot operator: 

# Method syntax: object.method()
line = file_object.readline()

# Function syntax: function(argument)
length = len(line)

Cleaning Up Lines

Use the strip method to remove whitespace (spaces, tabs, newlines): 

line = file_object.readline()
word = line.strip()

# Before strip: 'aah\n'
# After strip:  'aah'

Looping Through All Lines

You can use a file object directly in a for loop to process every line: 

for line in open('words.txt'):
    word = line.strip()
    print(word)

Updating Variables in Loops

Variables can be reassigned to new values. This is especially useful for keeping track of information as you loop through data.

Basic Variable Updates

You can assign a new value to a variable that already exists: 

x = 5
x
# 5

x = 7
x
# 7

x = x + 1
x
# 8

Initialize First

Before updating, create the variable:

z = 0
z = z + 1

Common Error

Don't update before initializing:

z = z + 1
# NameError!

Augmented Assignment Operators

Python provides shortcuts for common update operations: 

z = 0
z += 1    # Same as: z = z + 1
z += 2    # Same as: z = z + 2
z -= 1    # Same as: z = z - 1
z *= 3    # Same as: z = z * 3

📊 Terminology: Increasing a variable's value is called an increment. Decreasing it is called a decrement.

Counting with Loops

A counter is a variable used to count how many times something happens. Counters are essential for analyzing data in loops.

Counting All Words

Let's count how many words are in our word list: 

total = 0

for line in open('words.txt'):
    word = line.strip()
    total += 1

total
# 113783

The variable total is a counter that starts at 0 and increases by 1 for each word.

Counting Words with "e"

We can use multiple counters to track different things: 

total = 0
count = 0

for line in open('words.txt'):
    word = line.strip()
    total = total + 1
    if has_e(word):
        count += 1

# How many words have "e"?
count
# 76162

# As a percentage
count / total * 100
# 66.94%

📈 Insight: About two-thirds of English words contain the letter "e"—no wonder it was so hard to write a novel without it!

Simplifying with the in Operator

Python provides a powerful in operator that checks whether a character (or substring) appears in a string. This can greatly simplify our code.

Basic in Operator Usage

The in operator returns True if the character is found, False otherwise: 

word = 'Gadsby'
'e' in word
# False

'a' in word
# True

Simplifying has_e

We can rewrite has_e using the in operator: 

# Version 1: Using in operator
def has_e(word):
    if 'E' in word or 'e' in word:
        return True
    else:
        return False

# Version 2: Even simpler!
def has_e(word):
    return 'E' in word or 'e' in word

Using the lower Method

The lower method converts all letters to lowercase, making our function even cleaner: 

word = 'Gadsby'
word.lower()
# 'gadsby'

# Simplest version of has_e
def has_e(word):
    return 'e' in word.lower()

has_e('Gadsby')  # False
has_e('Emma')    # True

⚡ Important: The lower method creates a new string—it doesn't modify the original. The value of word remains 'Gadsby', not 'gadsby'.

The Linear Search Pattern

Linear search is a fundamental programming pattern. It loops through a sequence and checks each element until it finds what it's looking for—or reaches the end.

Creating a General Search Function

Let's create uses_any, which checks if a word uses any of a given set of letters: 

def uses_any(word, letters):
    for letter in word.lower():
        if letter in letters.lower():
            return True
    return False

uses_any('banana', 'aeiou')  # True
uses_any('apple', 'xyz')     # False
uses_any('Banana', 'AEIOU')  # True (case-insensitive)

How Linear Search Works

Step 1: Loop through each element in the sequence

Step 2: Check if the current element matches what you're looking for

Step 3: If it matches, return True immediately

Step 4: If you finish the loop without finding it, return False

✓ Correct Pattern

for item in sequence:
    if condition:
        return True
return False

✗ Common Mistake

for item in sequence:
    if condition:
        return True
    else:
        return False

⚠️ Warning: Don't put both return statements inside the loop! The second return should only happen after checking all elements.

Testing with Doctest

Docstrings can do more than document your functions—they can also test them! This helps ensure your code works correctly.

Writing Doctests

Include test examples in your docstring using >>> prompts: 

def uses_any(word, letters):
    """Checks if a word uses any of a list of letters.
    
    >>> uses_any('banana', 'aeiou')
    True
    >>> uses_any('apple', 'xyz')
    False
    """
    for letter in word.lower():
        if letter in letters.lower():
            return True
    return False

Running Doctests

Use this helper function to run the tests in a docstring: 

from doctest import run_docstring_examples

def run_doctests(func):
    run_docstring_examples(func, globals(), name=func.__name__)

# Test the function
run_doctests(uses_any)
# If all tests pass, no output appears

When Tests Fail

If a test fails, you'll see detailed output showing what went wrong: 

**********************************************************************
File "__main__", line 4, in uses_any_incorrect
Failed example:
    uses_any_incorrect('banana', 'aeiou')
Expected:
    True
Got:
    False

✅ Best Practice: Write doctests for every function. They serve as both documentation and automated tests!

Real-World Application: Spelling Bee

The New York Times Spelling Bee is a word puzzle that uses all the concepts you've learned. Players must form words using seven letters, with one letter required.

The Rules

• Words must have at least four letters

• Words must use only the seven available letters

• Words must include the required letter

• Letters can be repeated

Example Puzzle

Available letters: A, C, D, L, O, R, T (with R required)

✓ Valid Words

• "color" (uses R, 5+ letters)

• "ratatat" (uses R, repeats allowed)

• "doctor" (uses R, 6 letters)

✗ Invalid Words

• "told" (no R)

• "rat" (only 3 letters)

• "bee" (uses E, not available)

Scoring System

• Four-letter words: 1 point each

• Longer words: 1 point per letter

• Pangrams (use all 7 letters): 7 bonus points!

# Scoring examples
"card"     → 1 point   (4-letter word)
"color"    → 5 points  (5 letters)
"cartload" → 15 points (8 letters + 7 bonus for pangram)
